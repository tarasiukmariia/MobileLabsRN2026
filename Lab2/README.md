# Лабораторна робота №2

## Тема

Побудова вкладеної навігації та оптимізація відображення великих списків
у React Native.

## Мета

- Вивчення навігації (Stack, Drawer)
- Реалізація вкладеної навігації
- Робота з FlatList та SectionList
- Оптимізація продуктивності

---

## Інструкція запуску

1. **Встановити Node.js** (якщо ще не встановлено).
2. **Встановити Expo CLI** (глобально):

```
   npm install -g expo-cli
```

3. **Встановити залежності:**

```
   npm install
```

4. **Запустити проєкт:**

```
   npx expo start
```

4.  **Відкрити:**

- Expo Go (скан QR)
- Android Emulator
- iOS Simulator

---

## Реалізований функціонал

### Навігація

- Drawer Navigator (бокове меню)
- Stack Navigator (екрани новин)
- Вкладена структура (Drawer → Stack)

### Список новин (FlatList)

- Віртуалізація списку
- Pull-to-refresh
- Infinite scroll
- Header / Footer / Separator
- Оптимізація:
  - initialNumToRender
  - maxToRenderPerBatch
  - windowSize

### Деталі новини

- Передача параметрів між екранами
- Динамічний заголовок

### Контакти (SectionList)

- Групування даних
- Заголовки секцій
- Роздільники

### Drawer Menu

- Кастомний компонент
- Аватар
- ПІБ
- Група
- Пункти меню
